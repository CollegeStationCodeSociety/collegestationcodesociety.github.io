College Station Code Society

The College Station Code Society (CS++) is a club of young programmers looking to exercise and develop their skills. Membership in the club is reserved for 13-18 year olds. CS++ emphasizes the individual coder's ability to work in community with others. Currently, members have the opportunity of a challenging group project that includes the participation of various skill levels: a chess game based on python. Participants in the current group project collaborate using GitHub, however no previous experience with python or GitHub is required. CS++ typically holds monthly evening meetings, and any interested non-members are encouraged to visit! See the contact page for more information.
